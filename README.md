# .github

このリポジトリは、kome-kome アカウント配下のリポジトリで共通利用する GitHub 設定ファイルを管理します。

## 含まれる設定

- Issue テンプレート
- Pull Request テンプレート
- コントリビューションガイド
- 行動規範
- セキュリティポリシー
- サポート情報

## Issue テンプレート

- Bug Report
- Feature Request
- Task
- Documentation
- Question

## 方針

Issue は、できるだけ以下を満たす形で作成します。

- 目的が明確である
- 再現手順または完了条件がある
- 単独で着手できる粒度である
- 必要な背景情報が記載されている
